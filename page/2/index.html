LeetCode-225-Implement Stack using Queues

发表于 2020-03-16 分类于 LeetCode

225. Implement Stack using Queues

Easy

Description

Go to LeetCode

Solution

用一个队列的队头表示栈顶，队尾表示栈底。第 k 个元素入栈的时候先将其从队尾入队，然后将前面的 k-1 个元素依次从队头出队并从队尾入队。由于队列是 FIFO 的数据结构，这样能够保持先入栈的元素始终更靠近队头，也即更靠近栈顶，恰好符合栈的定义。

此方法时间复杂度为 push $O(n)$ ，pop/top/empty $O(1)$ 。

231. Power of Two

Easy

Given an integer, write a function to determine if it is a power of two.

Example 1:

Example 2:

Example 3:

Solution

注意到 2 的幂的特点是其二进制表示中只有一个 1 ，其余数位全部为 0 ，可以用 n & n-1 的技巧来检验这一点：

  • 若 n 是 2 的幂，设其二进制中第 k 位为 1 ，则 n-1 的二进制必然是1 ~ k - 1位均为 1 ，其余位为0 。这样 n & n-1 必为 0 ；
  • 若 n > 0 不是 2 的幂，则其二进制中必然包含至少两个 1 ，设位次最小的两个 1 分别在 k, m 位 (k < m) ，则n-1 的二进制必然是1 ~ k - 1位均为 1，第 k 位为 0 ，第 m 位以及更高位的 1 均保持不变。这样 n & n-1 必不为 0 。
